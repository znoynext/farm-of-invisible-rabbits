# Farm of Invisible Rabbits

Интерактивное статическое SPA для анализа косвенных сигналов на ферме и оценки возможного количества невидимых кроликов.

**Статус:** в разработке.

## Технологии

React, TypeScript и Vite. Расчётная модель полностью детерминирована и отделена от пользовательского интерфейса.

## Локальный запуск

Требуется Node.js 24 и npm.

```bash
npm ci
npm run dev
```

Проверки проекта:

```bash
npm run lint
npm run test
npm run build
```
